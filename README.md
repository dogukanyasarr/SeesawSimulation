# Doğukan YAŞAR "Seesaw Simulation Project"

## Proje Tanıtım Video: https://www.youtube.com/watch?v=Ibi3w4EP4rg

## Proje Yapısı

```text
seesaw-simulation/
├── index.html
├── style.css
└── js/
    ├── main.js       # Ana uygulama akışı (event listener'lar, döngüler)
    ├── physic.js     # Fiziksel hesaplamalar (tork, moment, denge)
    └── function.js   # Görsel oluşturma ve yardımcı fonksiyonlar
```

---
# Kullanılan Teknolojiler

| Teknoloji            | Açıklama                                                                             |
| -------------------- | ------------------------------------------------------------------------------------ |
| **HTML5**            | Sayfa yapısı ve görsel öğeler                                                        |
| **CSS3**             | Arayüz tasarımı, animasyonlar ve responsive yapı                                     |
| **JavaScript (ES6)** | Simülasyon mantığı, fizik hesaplamaları ve etkileşimler                              |
| **Modüler JS**       | Kod düzeni için ayrı dosyalara ayrılmış yapı (`main.js`, `physic.js`, `function.js`) |
